What is java?

Java is an object -oriented programming language that was developed by james gosling at Sun Microsystem which is now acquried by Orcale, In today's time, java is one of the most diverse language. Java programming language is a high-level object-oriented programming language designed to have as few implementation dependencie as possible.

Key feature of java programming language:

Here are the important feature of the Java language.

  1. Java programming language is designed to build object-oriented applications.
  2. java is multithreaded language with automatic memory management.
  3. In java, there is a feature called WORA i.e write once run anywhere after completion.
  4. Java code can be run on any computin platfrom.
